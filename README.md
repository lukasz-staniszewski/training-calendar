#### ŁUKASZ STANISZEWSKI

![image](https://user-images.githubusercontent.com/59453698/110309909-34db2880-8002-11eb-99ec-c40ea3f4edcb.png)

Simple cpp/qt project implementing family sports calendar with options to:
+ login by username,
+ add own trainings to calendar,
+ delete own trainings from calendar,
+ show nearest personal trainings,
+ show everyone's trainings in given day.
